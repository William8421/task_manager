About Task-Manager

Welcome to Task-Manager, your go-to solution for efficient task management! Task-Manager is a robust web application built to streamline your daily tasks, enabling you to stay organized and focused on what matters most. With intuitive features and a user-friendly interface, Task-Manager empowers users to create and manage tasks effortlessly.

Key Features

User Authentication: Task-Manager provides secure user authentication functionality, allowing users to register an account and log in securely. With password hashing and encryption, your personal information remains protected at all times.

Profile Management: Users have the flexibility to update their profiles, enabling them to customize their information and preferences according to their needs.

Task Management: Task-Manager simplifies task management by offering a comprehensive suite of tools to create, update, and delete tasks with ease. Whether it's organizing daily chores or tracking project milestones, Task-Manager has you covered.

Technologies Used

Node.js/Express: Task-Manager's backend is powered by Node.js, a versatile JavaScript runtime, and Express, a minimalist web framework for Node.js. This powerful combination provides a robust foundation for building scalable and efficient server-side applications.

PostgreSQL: Task-Manager leverages PostgreSQL, a powerful open-source relational database management system, to store and manage user data securely.

Angular: Task-Manager's frontend is developed using Angular, a popular TypeScript-based web application framework maintained by Google. Angular's modular architecture and extensive tooling enable rapid development of dynamic and responsive user interfaces.

Docker: Task-Manager utilizes Docker for production, enabling seamless containerization and deployment of both backend and frontend components.

Amazon Web Services (AWS)

Amazon RDS: Task-Manager leverages Amazon RDS (Relational Database Service) for PostgreSQL to host its database securely in the cloud. With features such as automated backups and scalability, Amazon RDS simplifies database management and enhances application reliability.

Amazon EC2: Task-Manager utilizes Amazon EC2 (Elastic Compute Cloud) instances to host its backend server, ensuring high availability and scalability for handling user requests efficiently.

Amazon S3: Task-Manager stores its frontend assets, such as HTML, CSS, and JavaScript files, on Amazon S3 (Simple Storage Service), a highly scalable object storage service. Amazon S3 provides durable and cost-effective storage for static web content, enabling fast and reliable delivery to users worldwide.
